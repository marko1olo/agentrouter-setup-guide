Зачем нужен локальный прокси?

На серверах AgentRouter работает жесткий WAF, блокирующий сторонние приложения (ошибка 401 Unauthorized или 500 sensitive_words_detected на длинных промптах). Кроме того, на стороне AgentRouter сломан оригинальный Tool Calling для моделей Claude (сервер возвращает 500 ошибку паники).

Наш асинхронный прокси на FastAPI полностью и автоматически решает обе проблемы:

  • Обход WAF (Cyrillic-Bypass): Прокси прозрачно заменяет английские буквы c на русские с в запросе, обходя WAF, и возвращает их обратно в ответе. Код остается синтаксически идеальным.
  • Мост Anthropic → OpenAI: Запросы автоматически переводятся в OpenAI формат и отправляются на стабильные модели gpt-5.5 или glm-5.2. Ваш редактор думает, что работает с Claude, и весь функционал (включая чтение/запись файлов) работает без сбоев.

Инструкция по установке и запуску:

  1. Установите зависимости: pip install fastapi uvicorn httpx
  2. Скачайте оба скрипта из репозитория и положите в одну папку: agentrouter_proxy.py и format_bridge.py
  3. Запустите прокси: python agentrouter_proxy.py

Как переключать модели (gpt-5.5 / glm-5.2):

Вы можете задать целевую модель через переменную среды AGENTROUTER_BRIDGE_MODEL при запуске:

# Windows (PowerShell):
$env:AGENTROUTER_BRIDGE_MODEL="glm-5.2"   # для дешевого и быстрого GLM
$env:AGENTROUTER_BRIDGE_MODEL="gpt-5.5"   # для мощного GPT (по умолчанию)
python agentrouter_proxy.py

# Linux / macOS (Bash):
export AGENTROUTER_BRIDGE_MODEL="glm-5.2"
python agentrouter_proxy.py

Победа над ошибкой "sensitive words detected"

Это фильтр самого AgentRouter. Он срабатывает на подозрительные слова в карточке персонажа или системном промте.

  • Замена тегов: Фильтр жестко триггерится на закрывающий XML-тег </character>. Замените его в карточке на </char> или </ch>, и 90% блокировок сразу пропадут!
  • Удаление триггеров: Уберите из карточек и промтов явные слова типа sex, drugs, violence, заменив их на синонимы или описательные конструкции.
  • Дополнительные параметры: В параметрах подключения в Таверне обязательно переопределите параметры temperature или top_p.

Фильтры цензуры моделей Claude

С взрослыми персонажами (NSFW) у Claude Opus/Sonnet на AgentRouter проблем практически нет. Однако для пробива жесткой цензуры используйте джейлбрейки (jailbreaks) и правильный контекст персонажа.
